🐧 Como o curso funciona

📖 Estrutura geral

Leia a teoria — cada módulo tem conteúdo detalhado, escrito como livro técnico, com tabelas, exemplos reais e comentários práticos.

Pratique no Linux em WebAssembly — o terminal à direita (ou abaixo no celular) abre um Linux isolado em WebAssembly/v86, rodando dentro do navegador, sem backend.

Responda as questões — cada módulo tem questões de múltipla escolha com comentários explicativos que aparecem após sua resposta.

Valide o módulo — execute os comandos no Linux WASM, marque o checklist do módulo e clique em "Validar módulo" para checar o checklist e o quiz.

Avance para o próximo — use os botões de navegação ou a barra de módulos no topo da página.

Desbloqueie o certificado — conclua todos os módulos e acerte ≥ 80% das questões totais. O certificado tem 84 horas e pode ser impresso.

💻 Terminal Linux

Use o terminal Linux em WebAssembly/v86 para praticar. O curso não mantém terminal paralelo; a liberação do módulo usa checklist local + quiz, pois o iframe WebAssembly não expõe o histórico real de comandos para o HTML pai.

🏆 Pontuação e certificado

  • Cada módulo exige que você execute os comandos pedidos nas tarefas e responda o quiz corretamente.
  • A nota mínima é 80% nas questões totais (soma de todos os módulos).
  • O progresso é salvo automaticamente no seu navegador (localStorage).
  • Se trocar de navegador ou apagar dados do navegador, o progresso é perdido.
  • O certificado pode ser baixado em PNG diretamente pelo navegador.

⚙️ Ambiente Linux em WASM

O terminal usa uma máquina Linux em WebAssembly no navegador. Para manter o curso frontend-only e sem backend, o ambiente principal é um Linux leve em v86. Os módulos continuam ensinando diferenças entre famílias Debian, Red Hat, Fedora e Arch, mas a execução prática ocorre no ambiente WASM disponível.

🔒 Segurança

O ambiente WASM roda isolado no navegador. Ele não usa backend e não executa comandos no sistema operacional real do aluno. Comandos destrutivos podem afetar apenas a máquina virtual/ambiente WASM carregado no browser.
