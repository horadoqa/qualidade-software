# 📊 Tipos de Report por Ferramenta

Cada ferramenta tem seu próprio “ecossistema de reporting”, e algumas dependem de extensões ou bibliotecas externas.

---

# 🤖 Robot Framework

### 📦 Tipos de report:

* **HTML Report (report.html)** ✅
* **Log.html (detalhado por keyword)**
* **Output.xml**
* **Allure Report (via plugin)**

### 📌 Destaques:

* Gera reports automaticamente ao final da execução
* Log detalhado por keyword (muito forte para análise)
* Muito usado em ambientes corporativos
