### 🥒 Cucumber

Ferramenta que suporta Behavior Driven Development (BDD).

**Características principais:**

* Escrita de testes em linguagem natural (Gherkin)
* Facilita colaboração entre times técnicos e de negócio
* Estrutura baseada em Given / When / Then
* Integração com várias linguagens (Java, JS, Ruby, etc.)
* Excelente para testes de aceitação e requisitos
