# 📊 Tipos de Report por Ferramenta

Cada ferramenta tem seu próprio “ecossistema de reporting”, e algumas dependem de extensões ou bibliotecas externas.

---

# 🥒 Cucumber

### 📦 Tipos de report:

* **HTML Report**
* **JSON Report**
* **JUnit XML**
* **Pretty Report**
* **Allure Report**

### 📌 Destaques:

* Relatórios baseados em cenários BDD (Given / When / Then)
* Foco em leitura por negócio
* Excelente rastreabilidade entre requisito e teste
