# 🛒 ServeRest.dev

O **ServeRest** é uma API REST gratuita que simula uma loja virtual, desenvolvida com o objetivo de servir como material de estudo para testes de API.

---

## 🧰 Ferramentas recomendadas

Você pode utilizar diversas ferramentas para interagir e testar a API, como:

* `cURL`
* `Postman`
* `k6`

Essas ferramentas permitem realizar desde testes manuais até automações e testes de performance.

---

## 🌐 Interface Web (Front-end)

O ServeRest também disponibiliza uma interface web (atualmente em versão beta), que facilita a exploração e execução de testes manuais:

👉 [https://front.serverest.dev/](https://front.serverest.dev/)

Com ela, é possível simular o uso da aplicação como um usuário final, complementando os testes automatizados.

---

## ⚠️ Testes de Performance

É possível executar testes de performance (carga, stress, etc.) na API.
No entanto, é importante seguir as recomendações oficiais do projeto para evitar impactos indevidos no ambiente compartilhado:

📄 [https://github.com/ServeRest/ServeRest#teste-de-carga](https://github.com/ServeRest/ServeRest#teste-de-carga)

---

## 📌 Resumo

O ServeRest é uma excelente opção para:

* Aprender testes de API
* Praticar automação
* Executar testes de performance em um ambiente controlado
* Simular cenários reais de e-commerce
