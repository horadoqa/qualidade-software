# Como instalar e executar o pytest em Python.

---

## 📦 1. Pré-requisitos

Você precisa ter:

* **Python instalado**
* **pip** (gerenciador de pacotes do Python)

Verifique no terminal:

```bash id="chk1"
python --version
pip --version
```

---

## ⚙️ 2. Instalar o pytest

No terminal, execute:

```bash id="inst1"
pip install pytest
```

Se estiver usando Python 3 em alguns sistemas:

```bash id="inst2"
pip3 install pytest
```
