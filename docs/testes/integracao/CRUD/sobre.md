# CRUD

CRUD é um acrônimo muito comum na área de programação e bancos de dados. Ele representa as quatro operações básicas que você pode fazer com dados:

* **C** – *Create* (Criar)
* **R** – *Read* (Ler/Consultar)
* **U** – *Update* (Atualizar)
* **D** – *Delete* (Excluir)

### Explicando de forma simples:

Imagine um cadastro de usuários em um sistema:

* **Create (Criar)** → adicionar um novo usuário
* **Read (Ler)** → visualizar os dados de um usuário
* **Update (Atualizar)** → editar as informações do usuário
* **Delete (Excluir)** → remover o usuário do sistema

Essas operações são a base de praticamente qualquer sistema que manipula dados, seja um site, aplicativo ou API.

### Onde o CRUD aparece?

* Em bancos de dados (como MySQL, PostgreSQL, MongoDB)
* Em APIs (por exemplo, endpoints HTTP: POST, GET, PUT/PATCH, DELETE)
* Em sistemas web e aplicativos em geral
