# Insomnia

O **Insomnia** é uma ferramenta usada para testar APIs (REST, GraphQL etc.). Você pode instalá-lo facilmente dependendo do seu sistema operacional.

---

## 💻 Windows

1. Acesse o site oficial:
   [https://insomnia.rest/download](https://insomnia.rest/download)
2. Clique em **Download for Windows**
3. Abra o arquivo `.exe` baixado
4. Siga o assistente de instalação (Next → Install)
5. Abra o programa pelo menu Iniciar

---

## 🍎 macOS

**Opção 1 (mais simples):**

1. Vá em [https://insomnia.rest/download](https://insomnia.rest/download)
2. Baixe a versão para Mac
3. Abra o arquivo `.dmg`
4. Arraste o **Insomnia** para a pasta **Applications**

**Opção 2 (Homebrew):**

```bash
brew install --cask insomnia
```

---

## 🐧 Linux (Ubuntu/Debian)

### Via Snap (recomendado):

```bash
sudo snap install insomnia
```

### Ou via .deb:

1. Baixe o arquivo no site oficial
2. Instale com:

```bash
sudo dpkg -i Insomnia*.deb
sudo apt-get install -f
```

---

## 🚀 Depois de instalar

* Abra o Insomnia
* Crie um workspace
* Comece a testar APIs (GET, POST, PUT, DELETE)

---

## 📌 Alternativa importante

O Insomnia é o mesmo tipo de ferramenta que o Postman, então se você já usou ele antes, vai se sentir em casa.
