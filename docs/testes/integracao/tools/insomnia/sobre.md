# Insomnia

Insomnia é uma ferramenta usada para **testar, desenvolver e depurar APIs**, muito parecida com o Postman.

Ela permite que desenvolvedores enviem requisições HTTP e vejam as respostas de um servidor sem precisar criar uma interface (como frontend ou aplicativo).

---

### 🔧 O que você pode fazer com o Insomnia?

* Enviar requisições **GET, POST, PUT, DELETE**
* Testar APIs REST e também **GraphQL**
* Visualizar respostas (JSON, XML, texto)
* Organizar requisições em projetos
* Criar variáveis de ambiente (ex: dev, produção)
* Automatizar testes simples de API

---

### 📌 Exemplo prático

Se você tem uma API:

* `GET /produtos` → lista produtos
* `POST /produtos` → cria produto

No Insomnia você pode:

1. Escolher o método (ex: POST)
2. Colocar a URL da API
3. Enviar um JSON com dados do produto
4. Ver a resposta do servidor imediatamente

---

### ⚖️ Insomnia vs Postman

* **Insomnia**: mais leve, interface mais simples, muito popular com GraphQL
* **Postman**: mais completo, com mais recursos de automação e colaboração

---

### 🚀 Resumindo

O Insomnia é uma ferramenta de teste de APIs, usada para facilitar o desenvolvimento e evitar a necessidade de escrever código só para testar requisições.
