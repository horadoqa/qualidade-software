# 💻 Onde o cURL funciona?

* 🐧 **Linux** → geralmente já vem instalado por padrão
* 🍎 **macOS** → também já vem instalado por padrão
* 🪟 **Windows** → funciona, mas pode precisar ser instalado (ou já vem em versões mais recentes do Windows 10/11)

---

### 📌 Importante

Mesmo que o sistema não venha com o cURL instalado, você pode facilmente adicionar:

* Linux → via gerenciador de pacotes (`apt`, `yum`, etc.)
* macOS → via Homebrew (`brew install curl`)
* Windows → download oficial ou já incluído no PowerShell moderno

---

### 🧠 Resumo

O cURL não é exclusivo do Linux. Ele é uma ferramenta **universal de linha de comando para requisições HTTP e transferência de dados**, disponível em praticamente qualquer sistema operacional moderno.
