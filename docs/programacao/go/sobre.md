Go (também chamada de **Golang**) é uma linguagem de programação criada pelo Google em 2009 por Robert Griesemer, Rob Pike e Ken Thompson.

Ela foi desenvolvida para ser:

* simples
* rápida
* eficiente
* concorrente
* fácil de manter em projetos grandes

---

# Características principais do Go

## Simples de aprender

A sintaxe do Go é limpa e enxuta, parecida com C, mas muito mais moderna.

Exemplo:

```go id="57t7v8"
package main

import "fmt"

func main() {
    fmt.Println("Olá mundo")
}
```

---

## Muito rápido

Go é compilado diretamente para código de máquina, então seu desempenho é próximo de linguagens de baixo nível como:

* C
* C++

Mas com muito menos complexidade.

---

## Concorrência nativa

Um dos maiores diferenciais do Go é trabalhar facilmente com tarefas simultâneas usando:

* goroutines
* channels

Isso faz Go ser excelente para:

* APIs
* servidores
* microsserviços
* sistemas distribuídos
* cloud computing

---

## Compilação extremamente rápida

Projetos grandes compilam em segundos.

---

## Garbage Collector

Go gerencia memória automaticamente, evitando boa parte da complexidade do C/C++.

---

# Onde Go é usado

Go é muito usado em:

* backend web
* DevOps
* cloud
* containers
* redes
* automação
* infraestrutura

Ferramentas famosas feitas em Go:

* Docker
* Kubernetes
* Terraform
* Prometheus

---

# Vantagens do Go

* Sintaxe simples
* Excelente performance
* Fácil deploy (gera binário único)
* Forte para backend
* Ótimo para concorrência
* Biblioteca padrão poderosa
* Fácil manutenção

---

# Desvantagens

* Generics chegaram tarde
* Menos flexível que Python
* Menos sofisticado que Rust em segurança de memória
* Ecossistema frontend praticamente inexistente
