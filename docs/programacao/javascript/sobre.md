JavaScript é uma linguagem de programação usada principalmente para criar interatividade em sites e aplicações web.

Ela foi criada em 1995 por Brendan Eich e se tornou uma das tecnologias fundamentais da web moderna.

---

# O que é JavaScript, na prática?

JavaScript é a linguagem que faz páginas web “ganharem vida”.

Sem JavaScript, uma página é apenas estática (texto e imagens). Com JavaScript, ela pode:

* responder a cliques
* atualizar conteúdo sem recarregar
* validar formulários
* criar animações
* interagir com APIs
* construir aplicações completas

---

# Onde o JavaScript é usado

## No navegador (frontend)

Ele roda diretamente no navegador (Chrome, Firefox, etc.) para controlar a interface do usuário.

## No servidor (backend)

Com Node.js, JavaScript também pode rodar no servidor.

# Características principais

## Linguagem interpretada

O código é executado diretamente pelo navegador ou Node.js.

## Tipagem dinâmica

Você não precisa definir tipos:

```javascript id="js2"
let x = 10
x = "texto"
```

---

## Base da web moderna

JavaScript é usado junto com:

* HTML (estrutura)
* CSS (estilo)
* JavaScript (lógica/interação)

# O que você pode construir com JavaScript

## Websites interativos

* menus dinâmicos

# Ecossistema importante

JavaScript tem frameworks e ferramentas muito usadas:

* React → interfaces modernas
* Vue.js → frontend reativo
* Angular → aplicações corporativas
* Express.js → backend

---

# Vantagens do JavaScript

* roda em qualquer navegador
* muito usado no mercado
* aprende rápido o básico
* grande ecossistema
* permite full-stack (frontend + backend)

---

# Desvantagens

* comportamento inconsistente em alguns casos
* pode ficar complexo em projetos grandes
* evolução rápida exige atualização constante
