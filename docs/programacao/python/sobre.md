Python é uma linguagem de programação de alto nível, interpretada e de sintaxe simples, criada para ser fácil de ler e escrever.

Ela foi criada por Guido van Rossum e lançada em 1991. Hoje, é uma das linguagens mais populares do mundo.

---

# O que torna Python especial

Python foi projetada com uma ideia simples:

> “Código deve ser legível como inglês”

Por isso, ela é muito usada por iniciantes e também por grandes empresas.

---

# Características principais

## Simples e fácil de aprender

O código Python é curto e direto:

```python id="p1"
print("Olá mundo")
```

---

## Linguagem interpretada

Python não precisa ser compilado manualmente. O código é executado linha por linha por um interpretador.

---

## Tipagem dinâmica

Você não precisa declarar tipos:

```python id="p2"
x = 10
x = "texto"
```

---

## Multiplataforma

Python roda em:

* Windows
* Linux
* macOS
* servidores
* cloud

---
