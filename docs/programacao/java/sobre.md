Java é uma linguagem de programação criada pela [Oracle Corporation](https://www.oracle.com/java/?utm_source=chatgpt.com) (originalmente pela Sun Microsystems) em 1995.

Ela foi projetada para ser:

* orientada a objetos
* segura
* portátil
* robusta
* multiplataforma

O grande lema do Java é:

> “Write Once, Run Anywhere”
>
> “Escreva uma vez, execute em qualquer lugar”

Isso significa que um programa Java pode rodar em diferentes sistemas operacionais graças à Java Virtual Machine (JVM).

---

# Como o Java funciona

O código Java é escrito em arquivos `.java`.

Exemplo:

```java 
public class Main {
    public static void main(String[] args) {
        System.out.println("Olá mundo");
    }
}
```

Depois ele é compilado:

```bash 
javac Main.java
```

Isso gera um arquivo `.class` com bytecode.

A JVM executa esse bytecode:

```bash 
java Main
```
