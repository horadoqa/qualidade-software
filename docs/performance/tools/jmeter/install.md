# ⚙️ Instalação

## Pré-requisitos

* Java 8 ou superior instalado

## Passos

1. Baixe o JMeter no site oficial:
   [https://jmeter.apache.org/](https://jmeter.apache.org/)
2. Extraia o arquivo `.zip` ou `.tgz`
3. Execute:

   * **Windows:** `bin/jmeter.bat`
   * **Linux/Mac:** `bin/jmeter`
