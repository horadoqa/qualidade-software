### 🤖 Robot Framework

Framework genérico de automação baseado em palavras-chave (keyword-driven).

**Características principais:**

* Sintaxe simples e legível (fácil para não programadores)
* Altamente extensível com bibliotecas em Python e Java
* Suporte a testes de UI, API, banco de dados e mais
* Boa integração com Selenium e Appium
* Forte uso em testes de aceitação (ATDD)
