# 📊 Tipos de Report por Ferramenta

Cada ferramenta tem seu próprio “ecossistema de reporting”, e algumas dependem de extensões ou bibliotecas externas.

---

# 🎭 Playwright

### 📦 Tipos de report:

* **HTML Report (nativo)** ✅
* **JSON Report**
* **JUnit XML (CI/CD)**
* **Allure Report (integração externa)**

### 📌 Destaques:

* Já vem com report HTML interativo
* Mostra screenshots, vídeos e traces de falha
* Excelente para debugging de falhas
