### 🧭 Selenium

Um dos frameworks mais antigos e amplamente usados para automação de navegadores.

**Características principais:**

* Suporte a múltiplas linguagens (Java, Python, C#, JavaScript, etc.)
* Compatível com praticamente todos os navegadores
* Grande ecossistema e comunidade ativa
* Flexível para diferentes tipos de testes web
* Base para diversas outras ferramentas de automação
