# 📊 Tipos de Report por Ferramenta

Cada ferramenta tem seu próprio “ecossistema de reporting”, e algumas dependem de extensões ou bibliotecas externas.

---

# 🧭 Selenium

### 📦 Tipos de report (dependente de framework):

* **JUnit Report**
* **TestNG Report**
* **Extent Reports**
* **Allure Report**
* **HTML personalizado (via libs)**

### 📌 Destaques:

* Selenium NÃO gera report nativo
* Depende totalmente da stack usada junto (JUnit, Pytest, etc.)
* Muito flexível, mas exige configuração
