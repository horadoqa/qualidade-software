# 📊 Tipos de Report por Ferramenta

Cada ferramenta tem seu próprio “ecossistema de reporting”, e algumas dependem de extensões ou bibliotecas externas.

---

# 🧪 Cypress

### 📦 Tipos de report:

* **Test Runner (UI nativo)** ✅
* **Mocha Awesome Report**
* **JUnit XML**
* **Allure Report (plugin)**
* **Dashboard Service (Cypress Cloud)**

### 📌 Destaques:

* Interface visual em tempo real (principal diferencial)
* Debug extremamente detalhado (time travel)
* Cypress Cloud permite histórico de execuções
