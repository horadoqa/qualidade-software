# Linux

**Linux** é um sistema operacional — ou seja, o software principal que gerencia o hardware do computador e permite que você use programas, navegue na internet, edite arquivos, etc.

De forma simples, ele é uma alternativa a sistemas como o Windows e o macOS.

### 💡 O que torna o Linux especial?

* **Código aberto (open source):** qualquer pessoa pode ver, modificar e distribuir o código.
* **Gratuito:** a maioria das versões não custa nada.
* **Seguro:** menos vulnerável a vírus em comparação com outros sistemas.
* **Personalizável:** você pode adaptar quase tudo ao seu gosto.

### 🧠 Como ele funciona?

O coração do Linux é o chamado **kernel** (núcleo), que controla a comunicação entre o hardware e o software.
A partir dele, surgem várias “versões” chamadas **distribuições (distros)**.

### 📦 Exemplos de distribuições Linux

* Ubuntu (fácil para iniciantes)
* Debian (estável e confiável)
* Fedora (tecnologias mais recentes)
* Arch Linux (para usuários avançados)

### 🖥️ Onde o Linux é usado?

* Servidores da internet (a maioria dos sites roda em Linux)
* Smartphones (o Android é baseado em Linux)
* Computadores pessoais
* Supercomputadores
