# Oh My Zsh

O **Oh My Zsh** é um framework (um conjunto de configurações prontas) para o shell **Zsh**, que deixa o terminal muito mais poderoso, bonito e produtivo.

---

## 💡 Em termos simples

O Zsh já é um shell (tipo o bash), mas o `Oh My Zsh` transforma ele em algo muito mais completo, adicionando:

* Autocomplete inteligente
* Sugestões de comandos enquanto você digita
* Temas visuais (prompt bonito)
* Plugins úteis para desenvolvimento

---

## 🔌 O que ele adiciona na prática

Com Oh My Zsh, você ganha coisas como:

### ⚡ Plugins populares

* `git` → atalhos para comandos Git
* `z` → navegação rápida entre pastas
* `docker` → comandos simplificados
* `npm` → atalhos pra Node

### 🎨 Temas

Você pode mudar completamente a aparência do terminal. Um dos mais famosos é o **Powerlevel10k**, que mostra:

* branch do Git
* status de execução
* tempo de resposta
* ícones modernos

---

## 🆚 Zsh vs Bash

| Recurso      | Bash     | Zsh + Oh My Zsh      |
| ------------ | -------- | -------------------- |
| Autocomplete | básico   | avançado             |
| Sugestões    | ❌        | ✅                    |
| Plugins      | ❌        | ✅                    |
| Temas        | limitado | muito personalizável |

---

## 🛠️ Como instalar (no WSL ou Linux)

Se você já tem o Zsh instalado:

### 1. Instalar o Zsh (se não tiver)

```bash
sudo apt install zsh
```

### 2. Instalar o Oh My Zsh

```bash
sh -c "$(curl -fsSL https://raw.githubusercontent.com/ohmyzsh/ohmyzsh/master/tools/install.sh)"
```

---

## 🚀 Vale a pena?

Se você usa terminal com frequência (principalmente com WSL), sim — faz muita diferença na produtividade e na experiência.

Mas vale um aviso:
Ele é mais sobre **conveniência e UX** do que desempenho puro. Não é obrigatório, mas depois que você acostuma, fica difícil voltar atrás.
