# Comandos

Os comandos do Linux servem para você interagir diretamente com o sistema operacional através do terminal (linha de comando). Eles permitem controlar praticamente tudo no sistema — desde arquivos até processos, usuários e rede — de forma rápida e poderosa.

Em vez de usar interface gráfica (cliques), você digita instruções e o sistema executa.

Aqui estão os [principais comandos do Linux](https://marianabsctba.github.io/Linux_Cheat_Sheet/), os mais usados no dia a dia no terminal.
