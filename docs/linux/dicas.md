# 🧠 1. Aprenda o básico do terminal (de verdade)

Não adianta decorar comando solto. Entenda como eles funcionam juntos:

* Use `man` (manual): `man ls`
* Use `--help`: `cp --help`

👉 Isso te ensina a se virar sozinho.

---

### ⚡ 2. Use TAB para autocompletar

Digite parte de um comando ou arquivo e aperte `TAB`.

Exemplo:

```bash
cd Doc[TAB]
```

👉 Economiza tempo e evita erro de digitação.

---

### 🔁 3. Reaproveite comandos antigos

* `↑` (seta pra cima) → mostra comandos anteriores
* `history` → lista tudo que você já digitou

👉 Você não precisa escrever tudo de novo.

---

### 📂 4. Entenda bem o sistema de diretórios

Tudo no Linux gira em torno disso:

* `/` → raiz
* `/home` → seus arquivos
* `/etc` → configurações

👉 Saber isso evita bagunça e erros.

---

### 🔒 5. Cuidado com o `sudo`

`sudo` te dá poder total no sistema.

Exemplo perigoso:

```bash
sudo rm -rf /
```

👉 Pode apagar tudo. Use com consciência.

---

### 🧪 6. Teste antes de fazer algo crítico

Antes de apagar ou mover muita coisa:

* Use `ls` para conferir
* Use `cp` antes de `mv` se estiver inseguro

👉 Evita dor de cabeça.

---

### 🔍 7. Use pipes (`|`) e redirecionamento

Isso é o “poder oculto” do Linux:

Exemplo:

```bash
ls | grep arquivo
```

👉 Combina comandos e filtra resultados.

---

### 📦 8. Mantenha o sistema atualizado

Depende da distro, mas geralmente:

```bash
sudo apt update && sudo apt upgrade
```

👉 Segurança e desempenho.

---

### 🧰 9. Instale ferramentas úteis

Algumas que valem muito a pena:

* `htop` → monitor de sistema melhor
* `neofetch` → info do sistema bonita
* `git` → controle de versão

---

### ⌨️ 10. Aprenda atalhos do terminal

* `Ctrl + C` → cancela comando
* `Ctrl + L` → limpa tela
* `Ctrl + D` → encerra sessão

👉 Parece pequeno, mas acelera muito.

---

### 💡 Dica final (a mais importante)

No Linux, você aprende fazendo.
Quebrou algo? Ótimo — você vai entender como funciona ao consertar.
