# Google Cloud Platform (GCP)

**gcloud** é a ferramenta de linha de comando da **Google Cloud Platform (GCP)**, usada para **criar, gerenciar e automatizar recursos na nuvem do Google**.

Em outras palavras, é como um “painel de controle no terminal” para você trabalhar com a infraestrutura da Google Cloud sem precisar usar o navegador.

---

## ☁️ O que é a Google Cloud?

A plataforma de nuvem do Google, chamada Google Cloud, oferece serviços como:

* servidores virtuais
* bancos de dados
* armazenamento
* inteligência artificial
* redes e APIs

---

## ⚙️ O que é o gcloud na prática?

O **gcloud** é parte do SDK do Google Cloud e permite que você:

* Criar máquinas virtuais
* Gerenciar clusters (Kubernetes)
* Fazer deploy de aplicações
* Configurar redes e permissões
* Automatizar tarefas na nuvem

Tudo isso usando comandos no terminal.

---

## 💻 Exemplo de uso

```bash id="gcloud1"
gcloud compute instances create minha-vm
```

Esse comando cria uma máquina virtual na Google Cloud.

Outro exemplo:

```bash id="gcloud2"
gcloud auth login
```

Esse comando faz login na sua conta Google Cloud.

---

## 🧠 Ideia simples

Pense assim:

* Interface web → clicar em botões
* gcloud → fazer tudo isso digitando comandos

---

## 🚀 Vantagens do gcloud

* ✔️ Automatização fácil via scripts
* ✔️ Controle total da infraestrutura
* ✔️ Ideal para DevOps e engenharia de cloud
* ✔️ Integração com CI/CD

---

## 💡 Resumo rápido

**gcloud é a ferramenta de terminal usada para controlar a Google Cloud**, permitindo gerenciar infraestrutura sem usar interface gráfica.
