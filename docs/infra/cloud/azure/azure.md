# Azure

**Azure** é a plataforma de **computação em nuvem da Microsoft**, usada para criar, hospedar e gerenciar serviços de TI pela internet.

Em vez de manter servidores físicos, empresas usam o Azure para “alugar” recursos como servidores, bancos de dados e redes, pagando conforme o uso.

---

## ☁️ O que a Azure oferece?

A Microsoft Azure fornece uma grande variedade de serviços, como:

* 🖥️ Máquinas virtuais (servidores na nuvem)
* 🗄️ Bancos de dados
* 💾 Armazenamento de arquivos
* 🌐 Redes e segurança
* 🤖 Inteligência artificial
* 📱 Hospedagem de aplicações web e mobile

---

## ⚙️ Como funciona na prática?

Com o Azure, você pode:

* Criar um servidor em minutos
* Hospedar sites e APIs
* Rodar aplicações sem precisar de infraestrutura própria
* Escalar recursos automaticamente quando aumenta o uso

---

## 💻 Exemplo simples de uso

Você pode criar uma máquina virtual na Azure usando ferramentas de linha de comando ou painel web, por exemplo:

* Criar uma VM para rodar um site
* Configurar um banco de dados para um app
* Publicar uma API globalmente

---

## 🧠 Ideia simples

* Antes: comprar e manter servidores físicos
* Com Azure: usar servidores pela internet sob demanda

---

## 🚀 Vantagens do Azure

* ✔️ Escalável (cresce conforme a necessidade)
* ✔️ Pagamento por uso
* ✔️ Integração com produtos Microsoft (Windows, Office, etc.)
* ✔️ Segurança corporativa forte
* ✔️ Usado por grandes empresas no mundo todo

---

## 🆚 Onde ele se encaixa?

O Azure compete diretamente com outras plataformas de nuvem como:

* Amazon Web Services (AWS)
* Google Cloud (GCP)

---

## 💡 Resumo rápido

**Azure é a plataforma de nuvem da Microsoft que permite criar e gerenciar sistemas, aplicativos e servidores sem precisar de infraestrutura física.**
