# AWS

**AWS** é a sigla para **Amazon Web Services**, que é uma plataforma de **serviços de computação em nuvem (cloud computing)** criada pela empresa Amazon.

Em vez de você precisar comprar e manter servidores físicos, a AWS permite que você **use recursos de TI pela internet**, pagando apenas pelo que usar.

---

## ☁️ O que a AWS oferece?

A AWS fornece vários serviços, como:

* 🖥️ **Servidores virtuais** (máquinas na nuvem)
* 💾 **Armazenamento de arquivos**
* 🗄️ **Bancos de dados**
* 🌐 **Redes e segurança**
* 🤖 Serviços de inteligência artificial
* 📦 Hospedagem de sites e aplicativos

---

## ⚙️ Exemplos de serviços famosos

* Amazon EC2 → cria servidores virtuais
* Amazon S3 → guarda arquivos como fotos, vídeos e backups
* AWS Lambda → executa código automaticamente sem gerenciar servidores

---

## 🧠 Ideia simples

Pense assim:

* Antes: você comprava um servidor físico e mantinha ele ligado
* Com AWS: você “aluga” servidores e recursos pela internet quando precisar

---

## 🚀 Vantagens da AWS

* ✔️ Escalável (cresce conforme a demanda)
* ✔️ Pagamento por uso
* ✔️ Alta disponibilidade
* ✔️ Usada no mundo inteiro
* ✔️ Reduz necessidade de infraestrutura física

---

## 🏢 Quem criou?

A AWS faz parte da empresa Amazon e é uma das maiores plataformas de nuvem do mundo.

---

## 💡 Resumo rápido

AWS é uma plataforma de **computação em nuvem** que permite usar servidores, armazenamento e outros serviços de TI pela internet, sem precisar ter infraestrutura própria.
