# 📋 Teste Funcional Manual

O **teste funcional manual** é baseado na execução de um roteiro de testes previamente definido, com o objetivo de validar se o sistema funciona conforme os requisitos.

Nesse tipo de teste, o tester segue passos específicos e compara o resultado obtido com o resultado esperado.

---

## 🎯 O que ele faz?

* Valida funcionalidades de acordo com requisitos
* Executa cenários passo a passo
* Garante que o sistema se comporta como esperado
* Identifica falhas em fluxos conhecidos

---

## 🧪 Exemplo

👉 Validar o fluxo de cadastro e login seguindo um passo a passo definido, verificando se cada etapa retorna o resultado esperado.

---

## 📌 Resumo

No teste funcional manual, o tester **segue um roteiro estruturado**, garantindo que as funcionalidades atendam aos requisitos especificados.
