# VS Code

O **VS Code** (ou Visual Studio Code) é um **editor de código-fonte** criado pela Microsoft. Ele serve para escrever programas em várias linguagens, como Python, JavaScript, C, Java, entre outras.

Ele não é exatamente um “programa completo para rodar sistemas” — é mais um **ambiente leve e muito poderoso para programar**.

---

## 💻 O que é o Visual Studio Code?

É um editor de código gratuito que permite:

* Escrever e editar código
* Executar programas (com extensões)
* Usar terminal integrado
* Depurar (debug) erros no código
* Instalar extensões para quase qualquer linguagem ou ferramenta

---

## 🚀 Por que ele é tão popular?

* 🧠 Leve e rápido
* 🔌 Muito personalizável (extensões)
* 🌐 Suporta várias linguagens
* 🧰 Terminal integrado (não precisa sair do programa)
* 🔍 Ajuda com autocompletar código (IntelliSense)

---

## 🧩 Exemplo de uso

Se você quiser programar em Python:

1. Abre o VS Code
2. Cria um arquivo `teste.py`
3. Escreve:

```python
print("Olá mundo")
```

4. Executa direto no editor

---

## 🔧 O que você pode fazer com ele?

* Criar sites (HTML, CSS, JavaScript)
* Programar scripts em Linux
* Desenvolver apps
* Trabalhar com Git e GitHub
* Automatizar tarefas

---

## 🧠 Resumo simples

👉 VS Code = um editor moderno para programar tudo em um só lugar.
