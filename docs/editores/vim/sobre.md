# VI ou VIM
 
O **vi** e o **vim** são editores de texto usados diretamente no terminal do Linux. Eles servem para criar e editar arquivos sem precisar de interface gráfica.

---

## 🧠 O que é o vi?

O **vi** é um editor de texto antigo e muito básico, presente em praticamente todos os sistemas Unix/Linux. Ele é leve, rápido e funciona só no terminal.

---

## 🚀 E o Vim?

O **vim** significa *“Vi Improved”* (Vi melhorado).
Ele é uma versão mais moderna e poderosa do vi, com mais recursos.

---

## 🔥 Para que eles servem?

* Editar arquivos de configuração do sistema
* Escrever scripts (como shell scripts)
* Trabalhar em servidores sem interface gráfica
* Fazer edições rápidas no terminal

---

## ⚙️ Como abrir um arquivo

```bash id="q8x9kd"
vim arquivo.txt
```

---

## 📌 Modos do Vim (isso é importante!)

O vim não funciona como um editor normal. Ele tem **modos**:

### 1. Modo normal (padrão)

* Usado para navegar e dar comandos

### 2. Modo inserção (digitar texto)

* Pressione:

```bash id="3v1m0p"
i
```

### 3. Modo comando

* Pressione:

```bash id="7x2lqp"
:
```

---

## 💾 Salvar e sair

Dentro do vim:

* Salvar:

```bash id="a1k9lm"
:w
```

* Sair:

```bash id="v8t3qn"
:q
```

* Salvar e sair:

```bash id="m2p7xz"
:wq
```

* Sair sem salvar:

```bash id="h6r9tw"
:q!
```

---

## ⚡ Por que alguém usa isso?

Mesmo parecendo difícil, ele é muito usado porque:

* Funciona em qualquer servidor Linux
* É extremamente rápido
* Não depende de interface gráfica
* Está sempre disponível

---

## 🧠 Resumo simples

* **vi** = editor básico antigo
* **vim** = versão melhorada e mais usada
* Ambos funcionam no terminal e são muito poderosos depois que você aprende
