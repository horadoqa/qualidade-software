# NANO

O **nano** é um **editor de texto simples que funciona no terminal do Linux**, feito para ser fácil de usar — especialmente para iniciantes.

---

## 🧠 O que é o GNU nano?

O **GNU nano** é um editor de texto em modo terminal que permite criar e editar arquivos de forma bem mais intuitiva do que editores como **vi/vim**.

Ele mostra na própria tela os comandos disponíveis, então você não precisa decorar muita coisa.

---

## 🚀 Para que ele serve?

* Editar arquivos de configuração
* Criar scripts (shell scripts, por exemplo)
* Fazer alterações rápidas em arquivos no Linux
* Usar em servidores sem interface gráfica

---

## ⚙️ Como abrir um arquivo

```bash id="nano_open"
nano arquivo.txt
```

Se o arquivo não existir, ele é criado automaticamente.

---

## 🧾 Como usar o nano

Dentro do nano, tudo é mais direto:

* Digitar texto normalmente (não tem “modo inserção” como no vim)
* Salvar:

```bash
Ctrl + O
```

* Sair:

```bash
Ctrl + X
```

---

## 📌 Atalhos importantes (aparecem na tela)

* `Ctrl + O` → salvar (Write Out)
* `Ctrl + X` → sair
* `Ctrl + K` → recortar linha
* `Ctrl + U` → colar
* `Ctrl + W` → buscar texto

---

## 🧠 Resumo rápido

👉 Nano é um editor de texto no terminal feito para ser fácil e direto, sem complicação.
