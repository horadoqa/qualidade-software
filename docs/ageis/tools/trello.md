## 📌 O que é o Trello?

O **Trello** é uma ferramenta de organização de tarefas baseada no método **Kanban**, usada para gerenciar projetos de forma visual, simples e colaborativa.

Ele funciona como um quadro digital onde você organiza o trabalho em **listas e cartões**, permitindo acompanhar o progresso das atividades de forma clara.

---

## 🧠 Como ele funciona

No Trello, tudo é organizado em três elementos principais:

* 📋 **Quadros (Boards)** → representam o projeto (ex: “Projeto Web”)
* 🗂️ **Listas (Lists)** → etapas do fluxo (ex: “A fazer”, “Fazendo”, “Feito”)
* 🃏 **Cartões (Cards)** → tarefas individuais (ex: “Criar API de login”)

---

## 🔁 Exemplo simples

Um projeto de aplicação web poderia ficar assim:

### 🟦 A fazer

* Criar banco de dados
* Implementar login

### 🟨 Fazendo

* Criar API de usuários

### 🟩 Feito

* Setup inicial do projeto

Os cartões vão sendo movidos entre as listas conforme o trabalho avança.

---

## ⚙️ Recursos do Trello

O Trello permite:

* 👥 Colaboração em equipe
* 📎 Anexar arquivos e links nos cartões
* 🗓️ Definir prazos (due dates)
* 🏷️ Usar etiquetas de prioridade ou categoria
* 💬 Comentários e comunicação dentro das tarefas
* 🔔 Notificações de atualização

---

## 💡 Onde ele é usado

* Projetos de software
* Organização de estudos
* Gestão de tarefas pessoais
* Planejamento de equipes de marketing, design e produto

---

## 🧠 Em resumo

O Trello é basicamente:

> Um quadro visual online para organizar tarefas de forma simples, usando o método Kanban.
