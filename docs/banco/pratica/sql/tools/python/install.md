# Instalação do Python

Instalar o **Python** é bem simples, mas o processo muda um pouco dependendo do seu sistema. Vou te guiar pelos principais:

---

## 💻 No Windows

1. Acesse o site oficial:
   👉 [https://www.python.org/downloads/](https://www.python.org/downloads/)

2. Clique em **Download Python** (vai baixar a versão mais recente)

3. Abra o instalador e **MUUUUITO importante**:
   ✔️ Marque a opção **"Add Python to PATH"**

4. Clique em **Install Now**

5. Após terminar, teste abrindo o Prompt de Comando e digitando:

   ```bash
   python --version
   ```

---

## 🍎 No macOS

1. Vá em: [https://www.python.org/downloads/mac-osx/](https://www.python.org/downloads/mac-osx/)

2. Baixe o instalador e execute

3. Depois, abra o Terminal e teste:

   ```bash
   python3 --version
   ```

---

## 🐧 No Linux (Ubuntu/Debian)

Normalmente já vem instalado, mas se não:

```bash
sudo apt update
sudo apt install python3
```

Depois teste:

```bash
python3 --version
```

---
