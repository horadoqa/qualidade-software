# Python

**Python** é uma linguagem de programação — ou seja, é uma forma de dar instruções para um computador fazer tarefas.

Ela é uma das linguagens mais populares do mundo porque é **simples de aprender** e muito **versátil**.

### 📌 Para que serve o Python?

Com Python, você pode:

* Criar sites e sistemas web
* Desenvolver jogos
* Fazer automações (tipo tarefas repetitivas)
* Trabalhar com **dados e inteligência artificial**
* Criar aplicativos e programas

### 💡 Por que tanta gente usa?

* Sintaxe simples (parece quase inglês)
* Grande comunidade (muita ajuda disponível)
* Funciona em várias áreas diferentes

### 🧠 Exemplo simples

Um código em Python pode ser assim:

```python
print("Olá, mundo!")
```

Isso faz o computador mostrar a mensagem **"Olá, mundo!"** na tela.
