# Robot Framework

O **Robot Framework** é uma ferramenta de automação usada principalmente para **testes de software**, mas também serve para automatizar tarefas repetitivas.

Ele usa uma linguagem bem simples e “legível”, parecida com frases em inglês, então é fácil de entender mesmo para iniciantes.

---

## 🤖 O que ele faz?

Com o Robot Framework você pode:

* Testar sites e aplicativos automaticamente
* Automatizar cliques, login e formulários
* Validar se sistemas estão funcionando corretamente
* Automatizar tarefas repetitivas no computador

---

## 🧠 Como ele funciona?

Em vez de escrever código complexo, você escreve algo assim:

```
*** Test Cases ***
Abrir Google
    Open Browser    https://google.com    chrome
    Input Text      name=q    Python
    Press Keys      name=q    ENTER
```

👉 Isso basicamente diz:

* abrir o Google
* digitar “Python”
* apertar Enter

---

## ⚙️ Ele usa Python por trás

O Robot Framework é escrito em Python e pode ser expandido com bibliotecas em Python.

Ou seja:

* Robot Framework = linguagem simples de automação
* Python = motor que roda tudo por baixo

---

## 📌 Onde ele é usado?

* Empresas de tecnologia (testes de sistemas)
* QA (Quality Assurance / testes de software)
* Automação de tarefas em sistemas web e desktop

---

## 👍 Por que ele é popular?

* Fácil de aprender
* Não exige programação avançada no começo
* Muito usado em testes automatizados profissionais
