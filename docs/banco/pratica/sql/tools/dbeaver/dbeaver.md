# DBEAVER

O **DBeaver** é uma ferramenta gráfica (GUI) para **gerenciar bancos de dados**. Ele funciona como um “painel de controle” onde você pode conectar, visualizar tabelas, executar SQL e administrar dados sem precisar ficar só no terminal.

---

## 🧠 O que o DBeaver faz?

Com o DBeaver você consegue:

* Conectar em vários bancos: PostgreSQL, MySQL, Oracle Database, SQL Server, entre outros
* Criar e editar tabelas
* Executar consultas SQL
* Visualizar dados de forma organizada
* Exportar/importar dados (CSV, Excel, etc.)
* Gerenciar usuários e permissões (dependendo do banco)

👉 Ele é muito usado por desenvolvedores, analistas de dados e DBAs.

---

## 💻 Como instalar o DBeaver

### 🔹 1. Baixar o instalador

Acesse o site oficial:
👉 [https://dbeaver.io/download/](https://dbeaver.io/download/)

Escolha a versão:

* **Community (gratuita)** ✅ recomendada para começar
* Enterprise (paga)

---

### 🔹 2. Instalar no seu sistema

#### 🪟 Windows

1. Baixe o `.exe`
2. Execute o instalador
3. Clique em *Next → Next → Install*
4. Finalize e abra o programa

---

#### 🐧 Linux (Ubuntu/Debian)

```bash
sudo apt update
sudo apt install dbeaver-ce
```

Ou baixe o `.deb` no site e instale manualmente:

```bash
sudo dpkg -i dbeaver-*.deb
```

---

#### 🍎 macOS

Opção 1 (recomendada com Homebrew):

```bash
brew install --cask dbeaver-community
```

Opção 2:

* Baixe o `.dmg` no site
* Arraste para Applications

---

## 🔌 Como usar pela primeira vez

1. Abra o DBeaver
2. Clique em **“Nova conexão”**
3. Escolha o banco (ex: PostgreSQL)
4. Preencha:

   * Host (ex: `localhost`)
   * Porta (ex: `5432`)
   * Usuário e senha
5. Teste a conexão ✅
6. Conecte

---

## 🚀 Resumindo

* O DBeaver é uma ferramenta visual para trabalhar com banco de dados
* Substitui (ou complementa) o uso de terminal
* Funciona com praticamente qualquer banco
* Fácil de instalar e usar
