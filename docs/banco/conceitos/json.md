# 📄 O que é JSON?

JSON é uma forma de organizar informações usando pares de **chave e valor**.

Exemplo:

```json
{
  "nome": "João",
  "idade": 25,
  "ativo": true
}
```

---

# 🧠 Explicação simples

Pense assim:

* 📦 JSON = uma “caixa de dados organizada”
* 🏷️ Cada informação tem um rótulo (chave) e um valor

---

# ⚙️ Para que o JSON é usado?

O JSON é usado para:

## 🌐 1. Troca de dados entre sistemas

Ex:

* APIs
* sites
* aplicativos

---

## 🗄️ 2. Armazenamento leve de dados

Alguns bancos NoSQL usam JSON, como o MongoDB.

---

## 📡 3. Comunicação frontend ↔ backend

Exemplo:

* site pede dados
* servidor responde em JSON

---

# 🧱 Estrutura do JSON

## 📌 1. Objeto

```json
{
  "nome": "Ana"
}
```

---

## 📌 2. Lista (array)

```json
{
  "nomes": ["Ana", "João", "Maria"]
}
```

---

## 📌 3. Tipos de dados

JSON aceita:

* texto → `"string"`
* número → `10`
* booleano → `true / false`
* lista → `[]`
* objeto → `{}`

---

# 🔄 JSON na prática

Exemplo de resposta de API:

```json
{
  "id": 1,
  "nome": "Carlos",
  "email": "carlos@email.com"
}
```

---

# 🧠 Analogia simples

Pense assim:

* 📄 JSON = ficha de cadastro
* 🧠 cada campo tem um nome e um valor

Ex:

* nome: João
* idade: 25
* ativo: sim

---

# ⚡ Por que JSON é importante?

## 🌍 1. Universal

Funciona em várias linguagens:

* JavaScript
* Python
* Java
* Go

---

## ⚡ 2. Leve

Mais simples que XML

---

## 🔗 3. Base de APIs modernas

Praticamente toda API usa JSON hoje

---

# 📌 Resumo

JSON é:

✔ um formato de dados
✔ baseado em chave e valor
✔ usado para troca de informações
✔ simples, leve e universal
