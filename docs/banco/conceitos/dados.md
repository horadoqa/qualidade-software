# Dados

**Dados** são qualquer **informação registrada que pode ser armazenada, processada ou transmitida por um sistema**.

👉 Em termos simples:
**dados são “fatos brutos” que ainda não foram interpretados.**

---

# 🧠 Explicação simples

Pense assim:

* 🔢 “25” → dado
* 📍 “25°C em Petrópolis hoje” → informação
* 🧠 “Está quente hoje, vou sair de camiseta” → conhecimento

---

# 📦 Exemplos de dados

* números: `10`, `3.14`, `2026`
* textos: `"João"`, `"erro no sistema"`
* imagens: foto de um rosto
* áudio: gravação de voz
* sensores: temperatura, localização
* logs de sistema (ex: Docker, servidores)

---

# ⚙️ Dados na computação

Na informática, dados são tudo o que um sistema:

* recebe
* armazena
* processa
* envia

Ex:

* um formulário preenchido em um site
* um pedido em um e-commerce
* informações de um container Docker

---

# 🧱 Dados no contexto de sistemas (ex: Docker)

Dentro de sistemas como o Docker, dados podem ser:

* arquivos de aplicação
* bancos de dados (ex: PostgreSQL)
* logs de execução
* configurações persistentes

👉 Por isso existem os **volumes**, para não perder esses dados quando containers param.

---

# 🧠 Dados vs Informação

| Dados        | Informação                |
| ------------ | ------------------------- |
| brutos       | interpretados             |
| sem contexto | com significado           |
| ex: “100”    | ex: “100 usuários online” |

---

# 🔄 Ciclo dos dados

1. 📥 Coleta (input)
2. 💾 Armazenamento
3. ⚙️ Processamento
4. 📤 Saída (informação)

---

# 📌 Resumo

Dados são:

✔ fatos brutos
✔ unidades básicas da informação
✔ qualquer coisa que possa ser armazenada ou processada
✔ base de tudo em sistemas digitais
